# v0.4.1 — QAM selection fix

Patch release fixing the Quick Access Menu game dropdown being effectively unusable while a game was running.

## Fixed

- **The QAM game dropdown no longer overrides your choice while a game is running.** Steam-context auto-selection re-applied the running game on every context update, so picking a different game from the dropdown reverted immediately and manual backup or restore targeted the wrong game. An explicit selection now takes precedence over auto-selection for as long as the panel stays open, and auto-selection resumes only once you clear the choice. The precedence rule is documented in `DEVELOPMENT.md`. (c6a80d4, 99b9a18)
