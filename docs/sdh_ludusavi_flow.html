2. Auto-Sync: Game Start (Restore)

Flow triggered when a Steam game is launched. The plugin checks if the backup is newer than the local save and restores if necessary.

sequenceDiagram participant ST as Steam participant F as Frontend participant B as Backend participant S as Service participant A as Adapter (pyludusavi) participant L as Ludusavi CLI ST->>F: AppLifetimeNotification (bRunning=true) F->>F: isTracked(game) opt If tracked and autoSync F->>F: showToast("Checking saves...") F->>B: handle_game_start(game, app_id) B->>S: handle_game_start() S->>S: _match_game() (SteamID/Alias/Name) S->>A: compare_recency(game) A->>L: ludusavi restore --preview L-->>A: "New" or "Different" A-->>S: "backup_newer" S->>S: _run_locked(restore) S->>A: restore(game) A->>L: ludusavi restore --force L-->>A: Success A-->>S: Result S->>S: _record_history() S-->>B: {status: "restored"} B-->>F: Result F->>F: showToast("Restore completed") end

3. Auto-Sync: Game Exit (Backup)

Flow triggered when a Steam game is closed. The plugin checks for changes and performs a backup if the local save has been updated.

sequenceDiagram participant ST as Steam participant F as Frontend participant B as Backend participant S as Service participant A as Adapter (pyludusavi) participant L as Ludusavi CLI ST->>F: AppLifetimeNotification (bRunning=false) opt If tracked and autoSync F->>F: showToast("Backing up saves...") F->>B: handle_game_exit(game, app_id) B->>S: handle_game_exit() S->>S: _match_game() S->>A: backup(game, preview=true) A->>L: ludusavi backup --preview L-->>A: Change: "Different" A-->>S: Preview Results S->>S: _run_locked(backup) S->>A: backup(game) A->>L: ludusavi backup --force L-->>A: Success A-->>S: Result S->>S: _record_history() S->>S: _refresh_statuses_unlocked() S-->>B: {status: "backed_up"} B-->>F: Result F->>F: showToast("Backup completed") end
