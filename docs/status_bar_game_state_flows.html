Status Bar Game State Flows

The status bar must describe the verified save state, not the work the plugin might do later. Both lifecycle paths begin with the same check state, then move to download, upload, current, unknown, or unable-to-sync copy after the backend has checked Ludusavi.

Launch Flow

Game start
App lifetime start

Steam reports the selected game has started.

VERIFYING GAME SAVE

Published before any restore language appears.

VERIFYING GAME SAVE
check_game_start

Compares local save recency against the existing backup.

Backup is newer

Call restore_game_on_start, then show backup restore action copy.

RESTORING BACKUP SAVE
Local save is current

No restore is needed. The strip reports the save is synchronized.

GAME SAVE UP TO DATE
No usable backup state

no_backup maps to unknown rather than pretending sync occurred.

UNKNOWN
Ambiguous recency

ambiguous_recency asks the user to choose a save source.

SAVE CONFLICT

Exit Flow

Game stop
App lifetime stop

Steam reports the selected game has exited.

VERIFYING GAME SAVE

Published before any backup language appears.

VERIFYING GAME SAVE
check_game_exit

Runs a Ludusavi backup preview without writing backup data.

Local save changed

Call backup_game_on_exit, then show backup action copy.

BACKING UP LOCAL SAVE
No local changes

local_current means there is no new save data to upload.

GAME SAVE UP TO DATE
No save files found

no_files_found and not_in_preview map to unknown.

UNKNOWN
Preview failed

preview_failed and game_error prevent sync.

UNABLE TO SYNC

Reason Mapping

Backend reason or result Status bar behavior Why
needed + restore RESTORING BACKUP SAVE The backup is newer and will be applied locally before launch.
needed + backup BACKING UP LOCAL SAVE Local save changes will be written to the backup target after exit.
ambiguous_recency SAVE CONFLICT The plugin needs a user decision before moving save data.
restored, backed_up, local_current GAME SAVE UP TO DATE The save state is current after action or no action was needed.
no_backup, no_files_found, not_in_preview UNKNOWN The plugin cannot prove a synchronized save state.
game_error, preview_failed UNABLE TO SYNC The check is unsafe or failed, so the plugin avoids save movement.
auto_sync_disabled, operation_running, unmatched_game, not_processed Hide the strip These are silent lifecycle skips rather than user-visible sync results.
